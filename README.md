IDE Console Battler Project C#


Installation & Start
Stellen Sie sicher, dass eine IDE wie Rider oder Visaul Studio auf Ihrem System installiert ist.

Starten Sie das Spiel per Doppelklick oder über eine IDE wie Rider:



Steuerung

Öffnen Sie die fighting_game.cs in Ihrer IDE 
Führen Sie die Main() auf Zeile 15 aus

Die Anwendung öffnet sich in der Konsole mit der Begrüßung Welcome To Slashers Keep

Wählen Sie einen von drei Charakteren mit 1, 2 oder 3

Benennen sie Ihren Charakter

Der Gegner wählt einen Zufallscharakter



Kampf (Rundenbasiert)

Wählen sie Ihren Wunschangriff. Folgen Sie den Eingabeaufforderungen 1 oder 2

Attack: Ein Standardangriff mit deiner ausgerüsteten Waffe.

Specialattack: Ein mächtiger Schlag mit höherem Schaden, aber geringerer Trefferchance. Bei Fehlschlag 0 Schaden



Features

Klassensystem: Wähle zwischen Warrior, Mage & Rogue (jeder Charakter hat mit eigene Stats).

Gegner-KI: Rundenbasierte Logik, bei der Gegner zufällig zwischen verschiedenen Angriffen wählen.


Spielregeln & Tipps

Die Kampfaktionen finden im Wechsel statt. 

Der Spieler führt seine Attacke zuerst aus dann der Gegner seine.

Siegbedingung: Bringe die HP deines Gegners auf Null. Sieg einer Runde

Spielabschlussbegingung: Gewiine oder Verliere 2 von 3 Runden gegen die KI
